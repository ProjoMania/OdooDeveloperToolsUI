About

Odoo Developer Tools is designed to make Odoo development and server management easier across Linux distributions.

This web application offers a modern interface for tasks previously handled through command-line tools.

Features:
  • SSH server configuration management
  • Odoo database operations (list, drop, restore)
  • Enterprise license extension
  • Responsive design that works across devices
